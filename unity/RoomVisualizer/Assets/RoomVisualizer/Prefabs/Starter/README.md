# Starter Prefabs

Place the 13 low-poly starter prefab assets here:

bed, desk, chair, wardrobe, shelves, poster, hook, guitar, window, rug, lamp, books, laptop

Each prefab should have:
- A single low-poly mesh with one material
- A PlaceableObject component pre-configured with AllowedSurfaces and grid footprint
- A 128x128 thumbnail sprite stored alongside the prefab

Floor-only prefabs: bed (2x3), desk (2x2), chair (1x1), wardrobe (2x2), shelves (1x2),
                    rug (3x2), lamp (1x1), guitar (1x2), books (1x1), laptop (1x1)
Wall-only prefabs:  poster (1x2), hook (1x1), window (2x2)
