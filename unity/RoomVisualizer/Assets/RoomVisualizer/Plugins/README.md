# Plugins

Place FsCheck DLL files here for property-based testing:

- `FsCheck.dll` (v3.x)
- `FsCheck.NUnit.dll` (v3.x)

Download from NuGet:
- https://www.nuget.org/packages/FsCheck/
- https://www.nuget.org/packages/FsCheck.NUnit/

A .nupkg file is a zip archive — rename to .zip, extract, and copy the net6.0 or netstandard2.1 DLL into this folder.
